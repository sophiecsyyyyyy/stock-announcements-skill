# stock-announcements-skill
一个基于专业指南的A股公告解读工具，输入公告文本，输出结构化解读。

## 使用方法

直接向AI发送一份A股上市公司公告原文，AI会按照以下框架自动解读：

1. 识别公告类型
2. 分析预期差
3. 三维基本面评估（盈利能力/成长性/风险）
4. 关键细节研判
5. 综合判断（短期/中长期）
6. 避坑提醒

## 安装方式

### 方式一：Claude / ChatGPT
复制 `SKILL.md` 中的内容作为系统指令，然后粘贴公告即可。

### 方式二：Dify / Coze
1. 创建Bot
2. 将 `SKILL.md` 作为系统提示词
3. 上传 `rules/` 下的文件作为知识库

### 方式三：本地部署
直接使用任何支持系统指令的LLM应用。

## 示例

**输入：
> XX科技业绩预告：预计2026年上半年净利润同比增长20%-30%，去年同期增长5%。

**输出：
> 📊 超预期利好（实际中值25% vs 预期？），短期偏多，建议关注是否为主业驱动...

## 规则来源

基于《A股市场上市公司公告解读指南》整理，涵盖：
- 业绩类、融资类、股权变动类、重组类、风险类等全部公告类型
- 预期差原则、三维评估、四大黄金法则
- 50+ 利好/利空判断规则
- 量化阈值标准
